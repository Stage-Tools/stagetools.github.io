SMPTEye

v0.18.4 · Windows 10/11

Reads LTC and MTC timecode simultaneously and displays them in a browser-based interface accessible from any device on the local network. No dedicated hardware required — just an audio or MIDI input.

Download .exe Manual
Overview
How it works
LTC support
Reads Linear Timecode via any audio interface with a line input. Signal level is monitored in real time with a visual indicator.
MTC support
Reads MIDI Timecode from any hardware or virtual MIDI port. Compatible with loopMIDI for software-based timecode sources.
Multi-source
Monitor multiple LTC and MTC sources simultaneously. Each source appears as a card with its own timecode display and status.
Browser-based UI
The interface runs in any browser on any device on the local network — laptop, tablet, phone. No app to install on remote devices.
Auto frame rate detection
Frame rate is detected automatically from the incoming signal. Supports 24, 25, 29.97 DF/NDF, and 30 fps.
Start with Windows
Optional autostart on boot. The server is ready before you open a browser — useful for always-on monitoring setups.
