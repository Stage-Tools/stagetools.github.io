PC Locker

v0.10.6 · Windows 10/11

A fullscreen overlay that instantly locks all keyboard input on every connected monitor — without closing or interrupting running applications. Designed for live shows, recording sessions, and unattended workstations.

Download .exe Manual
Overview
How it works
Fullscreen overlay
A semi-transparent overlay covers all connected monitors simultaneously. Underlying applications remain visible and keep running normally.
Keyboard lock
All keystrokes are blocked at the lowest Windows level — including Win key, Ctrl+Alt+Del, Ctrl+Shift+Esc, and Alt+Tab. The mouse remains free.
Multi-monitor
The overlay and unlock button appear on every connected display at the same time. No monitor is left unprotected.
Unlock methods
Unlock instantly with a configurable keyboard shortcut (default Ctrl+Alt+L), or via password dialog on any screen.
Configurable opacity
Adjust overlay darkness from barely visible (0.2) to nearly opaque (0.85). Default is 0.55 — desktop stays visible behind the lock.
No installation
Single portable .exe file. No installer, no dependencies. Drop it anywhere and run. Config is saved automatically on first launch.
